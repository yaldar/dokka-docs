Data class for the shape of a Ssimplified tweet A simplified tweet will be saved in the optional "replies array" field and/or the optional "parent tweet" field of a regular tweet A simplified tweet is the same as a regular tweet object except that the it's missing the replies array. This is so that a tweet object doesn't get too large when a reply thread grows in size
